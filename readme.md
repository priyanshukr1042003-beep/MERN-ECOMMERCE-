🛒 MERN E-Commerce Application

A full-stack e-commerce web application built using the MERN stack (MongoDB, Express.js, React, Node.js). This platform delivers a seamless online shopping experience for users and powerful management tools for admins.

The application leverages Redux Toolkit for efficient state management and Material UI for a modern, responsive, and intuitive user interface.

🚀 Features
👤 User Features
⭐ Product Reviews
Write, edit, and delete product reviews
Real-time updates on ratings and star distribution
❤️ Wishlist
Add and remove products from wishlist
Add personalized notes to saved items
📦 Order Management
Place new orders
View order history and details
👤 Profile Management
Update email and username
Manage multiple delivery addresses
🛍️ Shopping Cart
Add/remove products
Update quantities dynamically
View subtotal and pricing details
🛠️ Admin Features
📦 Product Management
Add, edit, delete, and soft-delete products
Manage product details like name, price, and stock
📑 Order Management
View all orders
Update order status (processing, shipped, delivered, etc.)
🔐 Security & Authentication
Secure login and signup system
OTP verification
Password reset functionality
Protected routes and logout system
🎨 User Experience
Clean and responsive UI with Material UI
Smooth navigation and intuitive design
Optimized for performance
📈 Scalability
Designed with scalable architecture
Capable of handling growing user base and data
